## Descripción de archivos. 

El archivo "authReducer" el que conteine las funcionalidades del contexto, ya sea para actualuzar la información 
como tambien para eliminar la misma. de esta forma se logra emular el funcionamiento del login.  

El archivo "AuthContext" será el que permita compartir la información asignada mediante el reducer a los componentes hijos,
dicha información será dispuesta dentro de las rutas definidas.
